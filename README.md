# 背包寵物
黏液科技的附加 繁體翻譯版

此附加靈感來源於著名的模組[InventoryPets](https://www.curseforge.com/minecraft/mc-mods/inventory-pets).<br>
它對黏液科技添加了類似於"寵物"的物品.<br>
如果你將背包寵物放在身上,它們可以免遭受到某類型的傷害,給你提供免費物品或藥水效果.<br>
但前提是你也要攜帶它們最愛的食物.
